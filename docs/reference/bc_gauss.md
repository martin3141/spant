# Apply and subtract a Gaussian smoother in the spectral domain.

Apply and subtract a Gaussian smoother in the spectral domain.

## Usage

``` r
bc_gauss(mrs_data, smo_ppm_sd)
```

## Arguments

- mrs_data:

  mrs_data object.

- smo_ppm_sd:

  Gaussian smoother sd in ppm units.

## Value

smoother subtracted data.
