# Set `mrs_data` object data points at the end of the FID to zero.

Set `mrs_data` object data points at the end of the FID to zero.

## Usage

``` r
zero_td_pts_end(mrs_data, pts)
```

## Arguments

- mrs_data:

  MRS data.

- pts:

  number of end points to set to zero.

## Value

modified `mrs_data` object.
