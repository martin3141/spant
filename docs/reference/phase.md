# Apply phasing parameters to MRS data.

Apply phasing parameters to MRS data.

## Usage

``` r
phase(mrs_data, zero_order, first_order = 0)
```

## Arguments

- mrs_data:

  MRS data.

- zero_order:

  zero'th order phase term in degrees.

- first_order:

  first order (frequency dependent) phase term in ms.

## Value

MRS data with applied phase parameters.
