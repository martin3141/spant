# Return a list of options for an ABfit analysis to maintain comparability with analyses performed with version 1.9.0 (and earlier) of spant.

Return a list of options for an ABfit analysis to maintain comparability
with analyses performed with version 1.9.0 (and earlier) of spant.

## Usage

``` r
abfit_opts_v1_9_0(...)
```

## Arguments

- ...:

  arguments passed to
  [abfit_opts](https://martin3141.github.io/spant/reference/abfit_opts.md).

## Value

full list of options.
