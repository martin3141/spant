# Estimate the correlation matrix for a basis set.

Estimate the correlation matrix for a basis set.

## Usage

``` r
calc_basis_corr_mat(basis, xlim = c(4, 0.2), zf = TRUE)
```

## Arguments

- basis:

  basis_set object.

- xlim:

  spectral range to use in ppm.

- zf:

  zero-fill the basis set.

## Value

correlation matrix.
