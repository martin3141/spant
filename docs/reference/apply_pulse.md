# Simulate an RF pulse on a single spin.

Simulate an RF pulse on a single spin.

## Usage

``` r
apply_pulse(sys, rho, spin_n, angle, nuc, xy)
```

## Arguments

- sys:

  spin system object.

- rho:

  density matrix.

- spin_n:

  spin index.

- angle:

  RF flip angle in degrees.

- nuc:

  nucleus influenced by the pulse.

- xy:

  x or y pulse.

## Value

density matrix.
