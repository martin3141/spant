# Convert mrs_data object to a vector.

Convert mrs_data object to a vector.

## Usage

``` r
mrs_data2vec(mrs_data, dyn = 1, x_pos = 1, y_pos = 1, z_pos = 1, coil = 1)
```

## Arguments

- mrs_data:

  MRS data object.

- dyn:

  dynamic index.

- x_pos:

  x index.

- y_pos:

  y index.

- z_pos:

  z index.

- coil:

  coil element index.

## Value

MRS data vector.
