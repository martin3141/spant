# Segment T1 weighted MRI data using the rpyANTs interface to ANTs and write to file.

Segment T1 weighted MRI data using the rpyANTs interface to ANTs and
write to file.

## Usage

``` r
segment_t1_rpyants(mri_path, out_dir = NULL)
```

## Arguments

- mri_path:

  path to the volumetric T1 data.

- out_dir:

  optional output directory. Defaults to the same directory as mri_path
  if not specified.
