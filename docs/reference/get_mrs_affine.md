# Generate an affine for nifti generation.

Generate an affine for nifti generation.

## Usage

``` r
get_mrs_affine(mrs_data, x_pos = 1, y_pos = 1, z_pos = 1)
```

## Arguments

- mrs_data:

  input data.

- x_pos:

  x_position coordinate.

- y_pos:

  y_position coordinate.

- z_pos:

  z_position coordinate.

## Value

affine matrix.
