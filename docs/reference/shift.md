# Apply a frequency shift to MRS data.

Apply a frequency shift to MRS data.

## Usage

``` r
shift(mrs_data, shift, units = "ppm")
```

## Arguments

- mrs_data:

  MRS data.

- shift:

  frequency shift (in ppm by default).

- units:

  of the shift ("ppm" or "hz").

## Value

frequency shifted MRS data.
