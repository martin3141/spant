# Zero all coherence orders other than the one supplied as an argument.

Zero all coherence orders other than the one supplied as an argument.

## Usage

``` r
coherence_filter(sys, rho, order = 0)
```

## Arguments

- sys:

  spin system object.

- rho:

  density matrix.

- order:

  coherence order to keep (default is 0).

## Value

density matrix.
