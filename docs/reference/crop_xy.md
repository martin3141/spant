# Crop an MRSI dataset in the x-y direction

Crop an MRSI dataset in the x-y direction

## Usage

``` r
crop_xy(mrs_data, x_dim, y_dim)
```

## Arguments

- mrs_data:

  MRS data object.

- x_dim:

  x dimension output length.

- y_dim:

  y dimension output length.

## Value

selected subset of MRS data.
