# Read the log from Dinesh's MoCo sLASER sequence.

Read the log from Dinesh's MoCo sLASER sequence.

## Usage

``` r
read_dkd_moco_log(path, date, end_time)
```

## Arguments

- path:

  path to the log file.

- date:

  scan date, eg "01-01-2025".

- end_time:

  scan time, eg "14:00".

## Value

data.frame of moco parameters.
