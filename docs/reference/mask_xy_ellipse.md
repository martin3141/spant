# Mask the voxels outside an elliptical region spanning the MRSI dataset in the x-y plane.

Mask the voxels outside an elliptical region spanning the MRSI dataset
in the x-y plane.

## Usage

``` r
mask_xy_ellipse(mrs_data)
```

## Arguments

- mrs_data:

  MRS data object.

## Value

masked MRS data.
