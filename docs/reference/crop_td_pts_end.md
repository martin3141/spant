# Crop `mrs_data` object data points at the end of the FID.

Crop `mrs_data` object data points at the end of the FID.

## Usage

``` r
crop_td_pts_end(mrs_data, pts)
```

## Arguments

- mrs_data:

  MRS data.

- pts:

  number of points to remove from the end of the FID.

## Value

cropped `mrs_data` object.
