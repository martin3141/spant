# Covert a linewidth in Hz to an equivalent alpha value in the time-domain ie: x \* exp(-t \* alpha).

Covert a linewidth in Hz to an equivalent alpha value in the time-domain
ie: x \* exp(-t \* alpha).

## Usage

``` r
lw2alpha(lw)
```

## Arguments

- lw:

  linewidth in Hz.

## Value

beta damping value.
