# Crop `mrs_data` object data points in the time-domain.

Crop `mrs_data` object data points in the time-domain.

## Usage

``` r
crop_td_pts(mrs_data, start = NULL, end = NULL)
```

## Arguments

- mrs_data:

  MRS data.

- start:

  starting data point (defaults to 1).

- end:

  ending data point (defaults to the last saved point).

## Value

cropped `mrs_data` object.
