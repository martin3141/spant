# Calculate the effective dimensions of a spline smoother from lambda.

Calculate the effective dimensions of a spline smoother from lambda.

## Usage

``` r
calc_ed_from_lambda(spline_basis, deriv_mat, lambda)
```

## Arguments

- spline_basis:

  spline basis.

- deriv_mat:

  derivative matrix.

- lambda:

  smoothing parameter.

## Value

the effective dimension value.
