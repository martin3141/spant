# Covert a beta value in the time-domain to an equivalent linewidth in Hz: x \* exp(-i \* t \* t \* beta).

Covert a beta value in the time-domain to an equivalent linewidth in Hz:
x \* exp(-i \* t \* t \* beta).

## Usage

``` r
beta2lw(beta)
```

## Arguments

- beta:

  beta damping value.

## Value

linewidth value in Hz.
