# Read a basis file in LCModel .basis format.

Read a basis file in LCModel .basis format.

## Usage

``` r
read_basis(basis_file, ref = def_ref(), sort_basis = TRUE)
```

## Arguments

- basis_file:

  path to basis file.

- ref:

  assumed ppm reference value.

- sort_basis:

  sort the basis set based on signal names.

## Value

basis object.
