# Return the frequency scale of an MRS dataset in Hz.

Return the frequency scale of an MRS dataset in Hz.

## Usage

``` r
hz(mrs_data, fs = NULL, N = NULL)
```

## Arguments

- mrs_data:

  MRS data.

- fs:

  sampling frequency in Hz.

- N:

  number of data points in the spectral dimension.

## Value

frequency scale.
