# Overwrite a subset of dynamic scans.

Overwrite a subset of dynamic scans.

## Usage

``` r
assign_dyns(mrs_data, subset, mrs_data_new)
```

## Arguments

- mrs_data:

  dynamic MRS data to be updated.

- subset:

  vector containing indices to the dynamic scans to be assigned.

- mrs_data_new:

  dynamic MRS data with new values.

## Value

MRS data with updated dynamics.
