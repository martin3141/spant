# Covert a linewidth in Hz to an equivalent beta value in the time-domain ie: x \* exp(-t \* t \* beta).

Covert a linewidth in Hz to an equivalent beta value in the time-domain
ie: x \* exp(-t \* t \* beta).

## Usage

``` r
lw2beta(lw)
```

## Arguments

- lw:

  linewidth in Hz.

## Value

beta damping value.
